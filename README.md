### Запуск и использование

Для старта запустить [Main](https://github.com/theojarrus/uni-links/blob/main/src/Main.java), после чего появится меню с выбором действий:

1. Инфо - показать информационное меню
2. Открыть ссылку - открыть сжатую ссылку
3. Сократить ссылку - создать сжатую ссылку
4. Управление ссылками - удалить/отредактировать существующую ссылку (требуется быть авторизованным в аккаунт владельца ссылки)
5. Войти в аккаунт - авторизироваться с помощью токена
6. Выйти из аккаунта - выйти из текущего аккаунта

### Архитектура

1. App - класс приложения, обьединяющий сценарии использования и cli
2. TokenHolder - компонент, отвечающий за состояние авторизации, использует TokenStorage
3. CliParser и CliHandler - делегаты, обрабатывающие ввод с консоли
4. LinkStorage - хранилище сжатых ссылок на базе hashmap, использует ViewStorage, TimestampChecker, TokenHolder для поддержки функций
5. TokenStorage - хранилище токенов, необходимо при создании новых пользователей и авторизации
6. ViewStorage - хранилище просмотров, необходимо для блокировки ссылок при превышении лимита
7. TimestampChecker - компонент создания и проверок временных меток
